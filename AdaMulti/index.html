Abstract


Multi-modal action recognition is a challenging task that requires effective fusion of information from different modalities such as RGB, depth, and skeleton data. In this research, we propose AdaMulti, an adaptive cascaded multi-modal recognition framework for sports action analysis. Our framework adaptively selects and combines the most informative modalities for different action categories, achieving state-of-the-art performance on multiple sports action recognition benchmarks. Experimental results demonstrate that AdaMulti significantly improves recognition accuracy and robustness compared to existing multi-modal fusion methods.

Methodology




AdaMulti consists of three main components: (1) adaptive modality selection, (2) cascaded feature extraction, and (3) multi-modal fusion. The system dynamically selects the optimal combination of modalities based on action characteristics, extracts hierarchical features, and fuses them using attention mechanisms.





Results





Our framework achieves competitive performance on multiple benchmarks, including NTU-RGB+D, Northwestern-UCLA, and MSR Action3D datasets. The adaptive modality selection mechanism effectively reduces computational cost while maintaining high recognition accuracy.





Applications




AdaMulti can be applied to various sports analysis scenarios, including real-time action recognition, performance assessment, and training feedback. The framework's adaptive nature makes it suitable for different sports with varying requirements and available sensors.
